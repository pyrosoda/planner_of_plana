Student eleph icon filenames should mirror `templates/students_portraits` slugs.

Naming rule:

- `Item_Icon_SecretStone_<portrait_slug>.png`

Examples:

- `Item_Icon_SecretStone_hifumi_swimsuit.png`
- `Item_Icon_SecretStone_toki_bunny_girl.png`
- `Item_Icon_SecretStone_tsubaki_guide.png`

This keeps student icon assets aligned with portrait/template names so future
planner lookups can key off the same slug.
