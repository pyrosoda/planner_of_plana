Inventory detail name templates live here.

Expected layout:

- `item_name_image_region.region.json`: item detail popup name area.
- `equip_name_image_region.region.json`: equipment detail popup name area.
- `<profile_id>/<item_id>.png`: cropped name image template for that item.
- `<profile_id>/<item_id>.json`: optional metadata for the name template.

Profile directories mirror `templates/inventory_detail`, for example
`tech_notes`, `tactical_bd`, `ooparts`, `activity_reports`, and `equipment`.
